1-3-1 뷰의 속성
뷰=안드로이드 각각의 화면 구성 요소(화면 안에 들어가는 것들)
뷰 그룹 = 일종의 그릇역할.(뷰를 담는 역할)/뷰 그룹이 뷰를 상속
위젯 = 뷰 중에서 눈에 보이는 것들(컨트롤). ex) 버튼, 텍스트...
레이아웃 = 위치를 결정해줌./ 레이아웃도 뷰이기 때문에 레이아웃 안에 레이아웃을 포함시킬 수 있음
상속
1. 자바는 모든 소스코드가 클래스 안에 들어가도록 되어있음.
2. 클래스 안에 메소드나 변수 (멤버)가 들어가게 됨.
3. 안드로이드의 화면 = 액티비티
*사람이 눈으로 보는 파일은 xml로 이루어짐.
*	<시작 태그
	속성1="속성값1"
	</끝태그>

<크기 지정 방법>
* layout_height(세로크기)와 layout_width(가로크기)는 무조건 넣어줘야 하는 속성
* Constraintlayout = 기본레이아웃
* match_parent 여유공간이 있다면 꽉채우게 하는것
* wrap_content 뷰에 들어있는 내용물의 크기에 따라 뷰의 크기가 결정됨
* 크기 값 지정 = 숫자를 넣기
*LinearLayout
*dp는 크기 지정시 붙이는 단위 ex) 200dp
* ctrl + z 는 뒤로가기
* app : 은 외부라이브러리에서 제공하는 속성을 쓸 때 사용
