- 과제
	- 사용자로부터 숫자를 입력 받는다.
	- 숫자를 구분한다.
	- 결과를 출력한다.
```Java
package test123;

import java.io.BufferedReader;//BufferedReader()을 사용하기 위해 필요
import java.io.InputStreamReader;//같은 이유

public class class123 {

	public static void main(String[] args) {
		try {//나중에 예외처리할 떄 배운다.
			// 사용자로부터 숫자를 입력받는다.
			InputStreamReader ir = new InputStreamReader(System.in);
			BufferedReader br = new BufferedReader(ir);
			System.out.print("숫자를 입력해주세요 : ");
			String str = br.readLine();//문자열로 받음
			
			int number = Integer.parseInt(str);//문자열이 숫자로만 이루어져 있다면, 이를 그대로 정수로 바꿔줌(문자가 섞여 있으면 오류)
			// 숫자를 구분한다.
			int flag = number % 3;
			//결과를 출력한다.
			if(flag == 0) {
				System.out.println("3의 배수입니다.");
			}else {
				System.out.println("3의 배수가 아닙니다.");
			}
		} catch(Exception e) {}
	}
}
```

### 19강 ~ 20강
1. switch ~ case문
	- 수식이나 변수의 값이 정수일 경우에만 사용이 가능하다.
	- 수식이나 변수의 값을 확인하여 값에 해당하는 부분으로 바로 이동하므로 비교하는데까지 걸리는 작업량이 많지가 않다.
- 과제
```Java
package test;

import java.io.BufferedReader;
import java.io.InputStreamReader;

public class Class123 {

	public static void main(String[] args) {
		try {
		InputStreamReader ir = new InputStreamReader(System.in);
		BufferedReader br = new BufferedReader(ir);
		
		System.out.print("첫번째 숫자를 입력해주세요 : ");
		String str1 = br.readLine();
		int number1 = Integer.parseInt(str1);
	
		System.out.print("두번째 숫자를 입력해주세요 : ");
		String str2 = br.readLine();
		int number2 = Integer.parseInt(str2);
		
		System.out.print("연산자를 입력해주세요 : ");
		char op = (char)System.in.read();
		System.in.read();
		System.in.read();
		
		switch(op) {
		case '+' :
			System.out.println(number1 + number2);
			break;
		case '-' :
			System.out.println(number1 - number2);
			break;
		case '*' :
			System.out.println(number1 * number2);
			break;
		case '/' :
			System.out.println(number1 / number2);
			break;
		}
		}catch(Exception e) {}
	}
}
```

### 21강 ~ 22강
1. for문 
```Java
package test;

//import java.io.BufferedReader;
//import java.io.InputStreamReader;

public class Class123 {

	public static void main(String[] args) {
		//이중 포문 이용 -> 구구단 출력
		for(int i = 1;i < 10;i++) {//자동 정렬할 수 있는 단축키 있는가?
			for(int k = 1;k < 10;k++) {
				System.out.printf(" %d * %d = %d", i, k, i*k);
			} 
			System.out.println();
		} 
	}
}
```
